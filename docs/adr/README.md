# Architecture Decision Records (ADR)

This directory stores architecture decisions for Libreroo.

## Naming

- File format: `NNNN-short-title.md`
- Example: `0002-introduce-openapi-client-generation.md`

## Status Values

- `proposed`
- `accepted`
- `superseded`
- `deprecated`

## Minimal ADR Template

```md
# NNNN - Title

- Date: YYYY-MM-DD
- Status: proposed|accepted|superseded|deprecated

## Context
What problem or forces led to this decision?

## Decision
What did we choose?

## Consequences
Positive outcomes, trade-offs, and constraints introduced.

## Alternatives Considered
Short list of options and why they were not chosen.
```
